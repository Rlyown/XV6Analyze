# 规则

## 文件改动规则
* 源码分析文件的命名规则为`<源码文件名>_<文件类型>.md`，如`main_c.md`
* 文件内部一定要在头部添加`[toc]`标签以生成目录
* 引用的图片放在统一放在`assets`目录下，在markdown文件内使用相对路径

## PR格式
* 目的/原因
* 描述（简述改动的内容）
